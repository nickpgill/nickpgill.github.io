Description

Wordle is a word-guessing game where players try to figure out a hidden word. They input guesses, and the game provides feedback on correct letters and their positions. Players aim to solve the puzzle within a set number of attempts by strategically deducing the concealed word.

Key Colors

The letter is correct and in its place.
The letter is correct, but not in its place.
The letter is wrong.
